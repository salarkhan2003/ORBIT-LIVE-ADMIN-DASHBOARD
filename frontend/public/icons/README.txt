This folder should contain bus icon images for OlaMapWrapper:
- bus-default.png (default bus icon, 64x64 or 48x48, transparent background)
- bus-critical.png (critical/alert bus icon, 64x64 or 48x48, transparent background)
- bus-ac.png (optional, AC bus icon, 64x64 or 48x48, transparent background)

Add your PNG or SVG icons here. All maps will use these for bus markers.
